# twowlaunchscript

This script fixes the white screen/EGL error on Wayland by preloading the system's native Wayland libraries.

1. cd into the same directory where turtlewowlauncher (not your home directory but a sub directory)
2. git clone this repo
3. chmod +x launch-turtlewow.bash or ./launch-turtlewow.fish depends which shell you are using echo $SHELL to find out)
4. ./launch-turtlewow.bash or ./launch-turtlewow.fish (depends which shell you are using echo $SHELL to find out)
